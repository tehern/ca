# calculator project
A module for doing calculations

## description
module allows user to run simple calculations, like add, subtract, multiply, and divide. This module was created for an assignment and used Visual Studio Code to create.

## instructions on how to run
1. Make sure that you have a codespace app installed on your computer.
2. Download or clone this repository.
3. Open the folder in a source code editor.
4. Run the file.

## features
can run simple calculations
asks if user wishes to give remainder form during division
implemented error handling
